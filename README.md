Atom _[CSON]_/JSON [_transpile_]r
=================================
Freely [_transpile_] between _[CSON]_ and JSON in [Atom].

[Another] plugin for _[atom-transpile]_.

Install
-------
`apm install transpile-cson` or search “cson” under Packages within Atom.
